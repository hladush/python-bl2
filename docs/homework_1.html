✏️ Homework 1: Variables & Data Types

Complete all tasks below. Write your code in a file called homework_1.py and run it to check your answers.

💡 Reminder: Use input() to read from the user, int() / float() to convert strings to numbers, and f-strings to build messages like f"Hello, {name}!".

🎯 Tasks

Task 1 — Personal Card

Ask the user to enter their name, age, and city.
Then print a formatted card like this:

--- My Info ---
Name: Alice
Age: 12
City: Kyiv
--------------

Task 2 — Rectangle Calculator

Ask the user to enter the width and height of a rectangle (as numbers).
Calculate and print:

  • The area (width × height)
  • The perimeter (2 × (width + height))

Task 3 — String Inspector

Ask the user to enter any word. Print:

  • The word in uppercase
  • The word in lowercase
  • How many characters it has
  • The first letter and the last letter (hint: use indexing — word[0] and word[-1])

Task 4 — Piggy Bank

Ask the user to enter the number of coins they have: 1 hryvnia coins, 2 hryvnia coins, and 5 hryvnia coins.
Calculate the total amount of money and print: "Total: X hryvnias".

Task 5 — Type Guesser ⭐

Ask the user to enter a value. Without using type(), try to figure out what type it could be and print your guess.

  • If it is "True" or "False" — print "This looks like a boolean!"
  • If it contains only digits — print "This looks like an integer!"
  • If it looks like a decimal number — print "This looks like a float!"
  • Otherwise — print "This looks like a string!"

Hints: str.isdigit() checks if all characters are digits. "." in text checks if there is a dot.
