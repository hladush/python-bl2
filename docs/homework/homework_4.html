✏️ Homework 4: Build & Deploy Your Homework Site

You have solved three Python homework sets — now it's time to show your work to the world! In this homework you will create a personal website that displays all your solutions and publish it live on GitHub Pages so anyone can visit it with a link.

💡 You already know HTML! This very site is built with HTML. Open any page, right-click anywhere, and choose "View Page Source" to see how it's made. Feel free to borrow ideas from it!

🎯 Tasks

Task 1 — Create a GitHub Repository

Create a free account on github.com if you don't have one yet.
Then create a new public repository named python-homework and initialize it with a README file.

💡 Hint
After logging in, click the + icon in the top-right corner → New repository. Set the name to python-homework, choose Public, tick "Add a README file", then click Create repository.

Task 2 — Build Your index.html

Create a file called index.html in your repository. It should have:

  • An <h1> with your name
  • A short intro paragraph: what you learned in this Python course
  • Three sections with headings: Homework 1, Homework 2, Homework 3 (the code will go inside them in the next task)
💡 Hint 1
Start with the HTML5 boilerplate: 
# This is HTML, not Python — but the structure is similar!
# Copy this into index.html:
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Python Homework</title>
</head>
<body>
  <h1>Alice's Python Homework</h1>
  <p>I learned Python this year!</p>
  <h2>Homework 1 — Variables</h2>
  <h2>Homework 2 — If-Else</h2>
  <h2>Homework 3 — Cycles</h2>
</body>
</html>
💡 Hint 2
You can create the file directly on GitHub: open your repository, click Add file → Create new file, type index.html as the name, paste your code, and click Commit changes.

Task 3 — Add Your HW1 & HW2 Solutions

Inside each homework section, add your Python solutions as syntax-highlighted code blocks using Prism.js (the same library this site uses). Add all 5 solutions from Homework 1 and all 5 from Homework 2.

💡 Hint 1
Add these two lines in the <head> to load Prism.js from CDN: 
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/themes/prism-tomorrow.min.css">
And add these two lines just before </body>: 
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/prism.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/components/prism-python.min.js"></script>
💡 Hint 2
Wrap each solution in a code block like this: 
<h3>Task 1 — Personal Card</h3>
<pre><code class="language-python">
name = input("Enter your name: ")
age  = input("Enter your age: ")
city = input("Enter your city: ")
print("--- My Info ---")
print(f"Name: {name}")
print(f"Age: {age}")
print(f"City: {city}")
print("--------------")
</code></pre>

Task 4 — Add HW3 Solutions & Style Your Page

Add all 5 solutions from Homework 3 in the same way.
Then make your page look nice by adding a <style> block in the <head> with at least these three styles:

  • A background color for the page
  • A font family for the text
  • A max-width and centered layout for the content
💡 Hint
Add a <style> block inside <head>: 
<style>
  body {
    background: #1e1e1e;
    color: #d4d4d4;
    font-family: 'Segoe UI', sans-serif;
  }
  .container {
    max-width: 860px;
    margin: 0 auto;
    padding: 24px;
  }
</style>
Then wrap all your content inside <div class="container">...</div>.

Task 5 — Deploy on GitHub Pages & Share Your Link ⭐

Publish your site so it's live on the internet with a real URL!

  • Go to your python-homework repository on GitHub
  • Open Settings → Pages
  • Under Source, select branch main and folder / (root)
  • Click Save — your site will be live at https://your-username.github.io/python-homework/
  • Add your live URL as an HTML comment at the top of index.html: <!-- My site: https://... -->
