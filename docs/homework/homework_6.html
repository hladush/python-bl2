✏️ Homework 6: Lists

Complete all tasks below. Write your code in homework_6.py and run it to check your answers.

💡 Quick reminder: Lists are your best friend for storing collections of data. These tasks combine lists with loops, functions, and conditionals — all the things you have learned so far!

🎯 Tasks

Task 1 — Sum and Average

Ask the user for 5 numbers (one at a time, using a loop). Store them in a list. Print the list, the sum of all numbers, and the average. Use a loop to calculate the sum — do not use Python's built-in sum() function.

💡 Hint 1
Use int(input(...)) inside a for loop and append() each number to your list.
💡 Hint 2
To find the average, divide the total by len(my_list).

Task 2 — Reverse Without Cheating

Write a function called reverse_list(lst) that takes a list and returns a new list with the items in reverse order. Do NOT use the built-in reverse() method or [::-1] slicing — build the reversed list yourself using a loop. Test it with at least two different lists.

💡 Hint 1
Create an empty list. Loop through the original and insert each item at position 0 using insert(0, item), or loop backwards using range(len(lst) - 1, -1, -1).
💡 Hint 2
Remember from Lecture 4: use return to send the new list back from your function.

Task 3 — Even and Odd Splitter

Create a list of 10 numbers between 1 and 100 (just type them in). Write code that builds two new lists: one containing all the even numbers, and one containing all the odd numbers. Print both lists and how many items are in each.

💡 Hint 1
A number is even if num % 2 == 0. Loop through the original list, check each number, and append() it to the right new list.
💡 Hint 2
Create both empty lists before the loop starts: evens = [] and odds = [].

Task 4 — Remove Duplicates

Write a function called remove_duplicates(lst) that takes a list and returns a new list with all duplicate values removed, keeping only the first occurrence of each item. For example, remove_duplicates([1, 2, 2, 3, 1, 4]) should return [1, 2, 3, 4]. Test it with a list of at least 8 items that has several duplicates.

💡 Hint 1
Create an empty result list. Loop through the original — only append an item if it is not already in the result list (use the in keyword to check).
💡 Hint 2
The check looks like: if item not in result: result.append(item).

Task 5 — Common and Unique Items

Write a function called common_items(list1, list2) that takes two lists and returns a new list containing only the items that appear in both lists (no duplicates in the result). Then write a second function called unique_items(list1, list2) that returns items that appear in only one of the two lists, but not both. Test both functions with lists of your friends' favorite games or movies.

💡 Hint
For common_items, loop through list1 and check if each item is also in list2. For unique_items, think about which items are in one list but not in the other.

Task 6 — Mini To-Do List App ⭐

Build a mini to-do list app. The program should loop forever (while True) and show a menu: 1) Add task, 2) Remove task, 3) Show all tasks, 4) Mark task as done (print it with a ✅ checkmark), 5) Quit. Use a list to store the tasks and another list to track which ones are done. Combine everything you know: lists, loops, functions, and if/else!
