Task 1 — Multiplication Table

Ask the user to enter a number. Print the full multiplication table for that number (1 to 10). Example for 7: "7 x 1 = 7" … "7 x 10 = 70".

💡 Hint
Use for i in range(1, 11) to loop from 1 to 10. Inside the loop: print(f"{n} x {i} = {n * i}").
✅ Show Solution 
n = int(input("Enter a number: "))

for i in range(1, 11):
    print(f"{n} x {i} = {n * i}")

Task 2 — Find the Biggest

Ask the user to enter 5 numbers one by one (using a loop). After all 5 are entered, print the largest number. Do not use max().

💡 Hint 1
Before the loop, set biggest = None. Inside the loop, update it: if biggest is None or num > biggest: biggest = num.
💡 Hint 2
Remember to convert the input to a number: num = float(input(...)).
✅ Show Solution 
biggest = None
for i in range(5):
    num = float(input(f"Enter number {i + 1}: "))
    if biggest is None or num > biggest:
        biggest = num

print(f"The biggest number is: {biggest}")
