✏️ Homework 5: Functions

Complete all tasks below. Write your code in homework_5.py and run it to check your answers.

💡 Quick reminder: Functions let you test each piece of your program in isolation. Write one function at a time and test it before moving on — this is how professionals work.

🎯 Tasks

Task 1 — Even or Odd Checker

Write a function is_even(n) that returns True if n is even, False otherwise. Ask the user for 5 numbers using a loop, call the function for each, and print "[n] is even" or "[n] is odd".

💡 Hint 1
Use n % 2 == 0 to check if a number is even.
💡 Hint 2
Collect the 5 numbers with for i in range(5) and input() inside.

Task 2 — FizzBuzz Function

Write a function fizzbuzz(n) that returns "Fizz" if n is divisible by 3, "Buzz" if divisible by 5, "FizzBuzz" if both, and the number as a string otherwise. Call it for 1–30 and print each result.

💡 Hint 1
Check n % 15 == 0 first (catches the both-divisible case), then 3, then 5.
💡 Hint 2
Return str(n) for the fallthrough case so all branches return a string.

Task 3 — Mini Statistics

Write three functions: my_min(lst), my_max(lst), and my_average(lst) — each takes a list of numbers. Do not use Python's built-in min(), max(), or sum(). Test with two different lists.

💡 Hint 1
For min/max, initialise with the first item: result = lst[0]. Loop through the rest comparing.
💡 Hint 2
For average: use your own summing loop, then divide by len(lst).

Task 4 — Recursive Countdown

Write a function countdown(n) that prints numbers from n down to 1 and then prints "Go!" — using recursion (the function calls itself) instead of a loop.

💡 Hint 1
Base case: if n == 0: print('Go!'); return.
💡 Hint 2
Recursive case: print n, then call countdown(n - 1).

Task 5 — Text Analyser ⭐

Write a function analyze(text) that accepts a string and returns a dictionary with keys: "words" (word count), "chars" (character count excluding spaces), "sentences" (count of . ? ! characters), and "longest_word". Test it on at least two different strings and print the results.
