✏️ Homework 2: If-Else Conditions

Complete all tasks below. Write your code in a file called homework_2.py and run it to check your answers.

💡 Quick reminder: Use if / elif / else for decisions. Combine conditions with and, or, not. Compare values with == != < > <= >=.

🎯 Tasks

Task 1 — BMI Calculator

Ask for weight (kg) and height (m). Calculate BMI = weight / height². Print the category: <18.5 → Underweight | 18.5–24.9 → Normal | 25–29.9 → Overweight | ≥30 → Obese.

💡 Hint 1
Use float(input(...)) for both weight and height since they may be decimals.
💡 Hint 2
Height squared: height ** 2 or height * height. So BMI = weight / height ** 2.

Task 2 — Cinema Ticket Price

A cinema charges: under 5 → free | 5–12 → 50 UAH | 13–17 → 80 UAH | 18–64 → 120 UAH | 65+ → 60 UAH. Ask for the user's age and print how much they pay.

💡 Hint
Use if / elif / elif / elif / else — one branch per age group. Check age ranges with age < 5, age <= 12, etc.

Task 3 — Rock, Paper, Scissors

Ask two players to enter their choice: "rock", "paper", or "scissors". Print who wins or if it is a draw.

💡 Hint 1
Check the draw case first: if p1 == p2: print("Draw!"). Then handle the winning combinations for Player 1.
💡 Hint 2
There are exactly 3 ways Player 1 wins: rock beats scissors, scissors beats paper, paper beats rock. Use or to combine them into one condition.

Task 4 — Traffic Light

Ask for a traffic light color: "red", "yellow", or "green". Print the driver instruction. Print "Unknown signal." for anything else.

💡 Hint
Compare strings directly: if color == "red". Use color.lower() first so it works for both "Red" and "red".

Task 5 — Password Checker ⭐

Ask the user to create a password. Check and print feedback for each rule:

  • At least 8 characters long
  • Contains at least one digit
  • Contains at least one uppercase letter

If all pass → "Strong password! ✅". Otherwise list what needs fixing.
