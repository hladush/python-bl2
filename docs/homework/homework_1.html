✏️ Homework 1: Variables & Data Types

Complete all tasks below. Write your code in a file called homework_1.py and run it to check your answers.

💡 Quick reminder: Use input() to read from the user, int() / float() to convert to numbers, and f-strings like f"Hello, {name}!" to build messages.

🎯 Tasks

Task 1 — Personal Card

Ask the user to enter their name, age, and city. Then print a formatted card:

--- My Info ---
Name: Alice
Age: 12
City: Kyiv
--------------
💡 Hint
Use three input() calls to get each piece of info, then print them with f-strings. The separator lines are just plain print("--- My Info ---").

Task 2 — Rectangle Calculator

Ask the user to enter the width and height of a rectangle (as numbers). Calculate and print the area (width × height) and the perimeter (2 × (width + height)).

💡 Hint 1
Use float(input(...)) to read a decimal number from the user.
💡 Hint 2
Area = width * height. Perimeter = 2 * (width + height).

Task 3 — String Inspector

Ask the user to enter any word. Print: the word in uppercase, in lowercase, its length, and the first and last letters.

💡 Hint
word[0] is the first character. word[-1] is the last character (negative index counts from the end). len(word) gives the total length.

Task 4 — Piggy Bank

Ask the user how many 1-hryvnia, 2-hryvnia, and 5-hryvnia coins they have. Calculate the total and print: "Total: X hryvnias".

💡 Hint
Total = coins_1 * 1 + coins_2 * 2 + coins_5 * 5. Make sure to convert the inputs to integers with int().

Task 5 — Type Guesser ⭐

Ask the user to enter a value. Without using type(), determine what type it looks like and print your guess:

  • "True" or "False" → "This looks like a boolean!"
  • Only digits → "This looks like an integer!"
  • Has a dot and digits → "This looks like a float!"
  • Anything else → "This looks like a string!"

Hints: text.isdigit() checks if all characters are digits. "." in text checks for a dot.
