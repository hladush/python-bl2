✏️ Homework 2: If-Else Conditions

Complete all tasks below. Write your code in a file called homework_2.py and run it to check your answers.

💡 Reminder: Use if / elif / else for decisions, comparison operators (== != < > <= >=), and logical operators (and, or, not) to combine conditions.

🎯 Tasks

Task 1 — Body Mass Index (BMI)

Ask the user for their weight (kg) and height (m). Calculate BMI using the formula: BMI = weight / height².
Print the category:

  • BMI < 18.5 → "Underweight"
  • 18.5 – 24.9 → "Normal weight"
  • 25 – 29.9 → "Overweight"
  • 30 or above → "Obese"

Task 2 — Ticket Price

A cinema charges different prices by age:

  • Under 5 → free
  • 5–12 → 50 hryvnias
  • 13–17 → 80 hryvnias
  • 18–64 → 120 hryvnias
  • 65 and over → 60 hryvnias

Ask for the user's age and print how much they pay.

Task 3 — Rock, Paper, Scissors

Ask two players to enter their choice: "rock", "paper", or "scissors".
Print who wins (Player 1, Player 2) or if it is a draw.
Rules: rock beats scissors, scissors beats paper, paper beats rock.

Task 4 — Traffic Light

Ask the user to enter a traffic light color: "red", "yellow", or "green".
Print the instruction for a driver:

  • red → "Stop!"
  • yellow → "Get ready…"
  • green → "Go!"
  • anything else → "Unknown signal."

Task 5 — Password Checker ⭐

Ask the user to create a password. Check all conditions and print feedback for each:

  • At least 8 characters long
  • Contains at least one digit (hint: loop through characters and check char.isdigit())
  • Contains at least one uppercase letter (hint: char.isupper())

If all conditions pass, print "Strong password! ✅". Otherwise list what needs to be fixed.
