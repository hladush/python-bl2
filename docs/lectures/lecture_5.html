📋 Lecture 5: Lists

A list lets you store many values in one variable, in order. Think of it like a backpack where you can toss in items, take them out, rearrange them, and check what you have at any time. Lists are one of the most important tools in programming — almost every real program uses them!

1. Creating and Indexing a List

You create a list with square brackets and separate items with commas. Each item has a position number called an index. The first item is at index 0 (not 1!). You can also count backwards from the end using negative numbers.

fruits = ['apple', 'banana', 'cherry']

print(fruits[0])   # apple  — first item (index 0)
print(fruits[1])   # banana — second item
print(fruits[-1])  # cherry — last item (negative counts from end)
print(fruits[-2])  # banana

# Change an item
fruits[1] = 'mango'
print(fruits)  # ['apple', 'mango', 'cherry']
💡 Tip: Python lists are zero-indexed: the first item is at index 0, not 1. This is true in almost every programming language — getting used to it now will save confusion later.

2. Slicing a List

Slicing lets you grab a chunk of a list at once. You write list[start:stop] — it gives you items from the start index up to (but not including) the stop index. You can leave out start or stop to mean "from the beginning" or "to the end".

numbers = [10, 20, 30, 40, 50]

print(numbers[1:3])   # [20, 30]  — index 1 up to (not including) 3
print(numbers[:3])    # [10, 20, 30] — from start up to index 3
print(numbers[2:])    # [30, 40, 50] — from index 2 to end
print(numbers[::2])   # [10, 30, 50] — every second item
print(numbers[::-1])  # [50, 40, 30, 20, 10] — reversed!

3. Common List Methods

Lists come with built-in methods — special commands you call with a dot. These let you add items, remove them, sort the list, and more. Here are the ones you will use most often.

items = ['cat', 'dog', 'bird']

items.append('fish')       # add to end
print(items)  # ['cat', 'dog', 'bird', 'fish']

items.insert(1, 'hamster') # insert at index 1
print(items)  # ['cat', 'hamster', 'dog', 'bird', 'fish']

items.remove('dog')        # remove by value
print(items)  # ['cat', 'hamster', 'bird', 'fish']

last = items.pop()         # remove and return last item
print(last)   # fish

numbers = [3, 1, 4, 1, 5]
numbers.sort()             # sort in place (ascending)
print(numbers)  # [1, 1, 3, 4, 5]
print(len(numbers))  # 5
💡 Tip: The sort() method changes the original list. If you want a sorted copy without changing the original, use sorted(my_list) instead.

4. Looping Over a List

You already know for loops from Lecture 3. They work perfectly with lists! You can loop through every item, or use enumerate() to get both the index and the value at the same time. A very common pattern is building a new list by looping through an old one.

scores = [85, 92, 78, 95, 60]

# Simple loop
for score in scores:
    print(score)

# enumerate() gives index and value together
for i, score in enumerate(scores):
    print(f'Student {i + 1}: {score}')

# Build a new list using a loop
doubled = []
for score in scores:
    doubled.append(score * 2)
print(doubled)  # [170, 184, 156, 190, 120]

5. Checking Membership and Counting

Want to know if something is in your list? Use the in keyword. It returns True or False, so you can use it directly in an if statement. You can also count how many times a value appears.

colors = ['red', 'green', 'blue']

if 'red' in colors:
    print('Red is in the list!')   # prints

if 'yellow' not in colors:
    print('Yellow is missing.')    # prints

# count() — how many times does a value appear?
numbers = [1, 2, 2, 3, 2, 4]
print(numbers.count(2))  # 3
💡 Tip: Use if item in my_list to check whether a value exists — no loop needed. It is fast to write and easy to read, which is exactly what good code looks like.
