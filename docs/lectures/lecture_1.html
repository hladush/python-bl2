📖 Lecture 1: Variables & Data Types

In this lecture you will learn how to store information in Python using variables, and what different data types look like.

1. What is a Variable?

A variable is like a labeled box — you give it a name and store a value inside. You can always look inside the box or change what's in it.

# Creating variables
name = "Alice"       # text
age  = 12            # whole number
height = 1.55        # decimal number
is_student = True    # yes / no

# Printing variables
print(name)          # Alice
print(age)           # 12
print(is_student)    # True
💡 Naming rules: Use lowercase letters and underscores: my_name, favorite_color. Variable names cannot start with a number and cannot have spaces.

2. Data Types

Every value in Python has a type. The four most common types are:

x = 42          # int   — whole numbers: 0, 1, -5, 100
y = 3.14        # float — decimal numbers: 1.5, -0.7, 3.14
z = "hello"     # str   — text in quotes: "hi", 'world'
flag = True     # bool  — only two values: True or False

# Check the type of any value with type()
print(type(x))     # <class 'int'>
print(type(y))     # <class 'float'>
print(type(z))     # <class 'str'>
print(type(flag))  # <class 'bool'>

3. Working with Numbers

a = 10
b = 3

print(a + b)   # 13  — addition
print(a - b)   # 7   — subtraction
print(a * b)   # 30  — multiplication
print(a / b)   # 3.333...  — division (always float)
print(a // b)  # 3   — floor division (whole part only)
print(a % b)   # 1   — remainder
print(a ** b)  # 1000 — power (10 to the 3rd)

4. Working with Strings

name = "Alice"

print(len(name))          # 5     — length
print(name.upper())       # ALICE — all uppercase
print(name.lower())       # alice — all lowercase

# Combining strings (concatenation)
greeting = "Hello, " + name + "!"
print(greeting)           # Hello, Alice!

# f-strings — the easiest way to insert variables into text
age = 12
print(f"My name is {name} and I am {age} years old.")

5. Converting Between Types

# String → Number
num = int("42")      # 42
dec = float("3.14")  # 3.14

# Number → String
text = str(100)      # "100"

# Useful when reading user input:
user_input = input("Enter a number: ")   # input() always returns a string
number = int(user_input)                  # convert to int before doing math
print(number + 5)
💡 Remember: input() always gives you a string. If you want to do math with it, convert it first with int() or float().
