📖 Lecture 3: Cycles (Loops)

A loop lets you run the same block of code many times without repeating yourself. Python has two types of loops: for and while.

1. The for Loop with range()

# Counting from 0 to 4
for i in range(5):
    print(i)
# Prints: 0  1  2  3  4

# range(start, stop) — from start up to (but not including) stop
for i in range(1, 6):
    print(i)
# Prints: 1  2  3  4  5

# range(start, stop, step) — every 2nd number
for i in range(0, 10, 2):
    print(i)
# Prints: 0  2  4  6  8

# Counting backwards
for i in range(5, 0, -1):
    print(i)
# Prints: 5  4  3  2  1

2. Looping Over a List

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
# apple
# banana
# cherry

# Loop with both index and value using enumerate()
for i, fruit in enumerate(fruits):
    print(f"{i + 1}. {fruit}")
# 1. apple
# 2. banana
# 3. cherry

3. The while Loop

A while loop keeps running as long as a condition is True.

count = 1
while count <= 5:
    print(count)
    count += 1   # count = count + 1
# Prints: 1  2  3  4  5

# Asking until the user enters the right answer
secret = "python"
guess = ""
while guess != secret:
    guess = input("Guess the secret word: ")
print("Correct! 🎉")
⚠️ Infinite loop! If the condition in a while loop never becomes False, the program runs forever. Always make sure something changes inside the loop!

4. break and continue

# break — stop the loop immediately
for i in range(10):
    if i == 5:
        break
    print(i)
# Prints: 0  1  2  3  4

# continue — skip the rest of this iteration, go to the next one
for i in range(10):
    if i % 2 == 0:
        continue   # skip even numbers
    print(i)
# Prints: 1  3  5  7  9

5. Nested Loops

# Drawing a triangle with stars
for i in range(1, 6):
    print("*" * i)
# *
# **
# ***
# ****
# *****

# Multiplication table (3x3)
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i*j}")
    print()   # blank line after each row

Task 1 — Count Down

Use a for loop to print numbers from 10 down to 1, then print "Blast off! 🚀".

💡 Hint
range(10, 0, -1) generates: 10, 9, 8 … 1. The third argument is the step — a negative step counts backwards.
✅ Show Solution 
for i in range(10, 0, -1):
    print(i)
print("Blast off! 🚀")

Task 2 — Sum of Numbers

Ask the user to enter a number n. Use a loop to calculate the sum of all numbers from 1 to n and print the result. Example: n = 5 → "Sum = 15". To check: enter 5.

💡 Hint 1
Create a total = 0 variable before the loop. Inside the loop, add each number to it: total += i.
💡 Hint 2
Use range(1, n + 1) — the stop value is exclusive, so you need n + 1 to include n itself.
✅ Show Solution 
n = int(input("Enter a number: "))
total = 0
for i in range(1, n + 1):
    total += i
print(f"Sum = {total}")

Task 3 — Favourite List

Create a list of 5 of your favourite things. Use a for loop to print each one with its position number. Example: "1. Pizza", "2. Minecraft", …

💡 Hint
enumerate(my_list) gives you both the index and the value at once: for i, item in enumerate(my_list). Add 1 to i so the count starts at 1.
✅ Show Solution 
favorites = ["Pizza", "Minecraft", "Music", "Football", "Books"]

for i, item in enumerate(favorites):
    print(f"{i + 1}. {item}")

Task 4 — Guess the Number

Hardcode secret = 7. Use a while loop to keep asking the user to guess. Print "Too high!", "Too low!", or "Correct! 🎉" after each guess. To check: enter 7 as your first guess.

💡 Hint 1
Use while True and break when the guess is correct. This keeps the loop running until you explicitly stop it.
💡 Hint 2
input() returns a string. Convert it to an integer first: guess = int(input("Your guess: ")).
✅ Show Solution 
secret = 7

while True:
    guess = int(input("Your guess: "))
    if guess > secret:
        print("Too high!")
    elif guess < secret:
        print("Too low!")
    else:
        print("Correct! 🎉")
        break
