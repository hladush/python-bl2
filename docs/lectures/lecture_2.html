📖 Lecture 2: If-Else Conditions

In this lecture you will learn how to make your program make decisions — doing different things depending on whether a condition is true or false.

1. The if Statement

An if statement runs a block of code only when a condition is True.

age = 15

if age >= 18:
    print("You are an adult.")
else:
    print("You are a child.")

# Output: You are a child.
💡 Indentation matters! The code inside if must be indented (4 spaces or 1 Tab). Python uses indentation to know which lines belong together.

2. Adding More Options with elif

Use elif (short for "else if") to check multiple conditions one after another.

score = 75

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F")

# Output: Grade: C

3. Comparison Operators

These operators compare two values and return True or False:

x = 10
y = 5

print(x == y)   # False  — equal to
print(x != y)   # True   — not equal to
print(x >  y)   # True   — greater than
print(x <  y)   # False  — less than
print(x >= y)   # True   — greater than or equal to
print(x <= y)   # False  — less than or equal to

4. Logical Operators: and, or, not

age = 14
has_ticket = True

# and — both conditions must be True
if age >= 12 and has_ticket:
    print("You can enter!")

# or — at least one condition must be True
is_weekend = False
is_holiday = True
if is_weekend or is_holiday:
    print("No school today!")

# not — flips True to False and False to True
is_raining = False
if not is_raining:
    print("Let's go outside!")

5. Nested Conditions

temperature = 25
is_sunny = True

if temperature > 20:
    if is_sunny:
        print("Great weather for a walk!")
    else:
        print("Warm but cloudy.")
else:
    print("Too cold to go outside.")

🎯 Tasks

Task 1 — Even or Odd

Ask the user to enter a number. Print "Even" if it is divisible by 2, or "Odd" otherwise.

💡 Hint
A number is even when there is no remainder after dividing by 2. Check with: if number % 2 == 0.

Task 2 — Grade Checker

Ask the user to enter a score (0–100). Print the letter grade: 90–100 → A, 80–89 → B, 70–79 → C, 60–69 → D, below 60 → F.

💡 Hint
Always check from the highest score first using if / elif / elif / else. Once a condition matches, Python skips all the rest, so order matters!

Task 3 — FizzBuzz

Given a number, print "FizzBuzz" if divisible by both 3 and 5, "Fizz" if by 3, "Buzz" if by 5, or the number itself otherwise. Test with: 15, 9, 10, 7.

💡 Hint 1
Check the "both 3 and 5" case first! If you check 3 first, you'll print "Fizz" for 15 and never reach "FizzBuzz".
💡 Hint 2
Use % to check divisibility: num % 3 == 0 means the number is divisible by 3 with no remainder.

Task 4 — Can I Enter?

Create two variables: age and has_ticket (True/False). Print "Welcome!" only if the person is at least 16 and has a ticket. Otherwise print a message explaining which condition failed.

💡 Hint
Use and to check both conditions at once: if age >= 16 and has_ticket. For the else, use two separate if statements to figure out which part failed.

Task 5 — Season Detector ⭐

Ask the user to enter a month number (1–12). Print the season: 12, 1, 2 → Winter | 3, 4, 5 → Spring | 6, 7, 8 → Summer | 9, 10, 11 → Autumn.
