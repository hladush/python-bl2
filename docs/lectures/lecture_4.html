🧩 Lecture 4: Functions

A function is a reusable block of code you give a name to. Instead of writing the same steps over and over, you write them once inside a function and call it whenever you need it.

🔧 Software Engineering Angle — DRY Principle
DRY stands for Don't Repeat Yourself. Functions are the first tool programmers use to avoid copy-pasting code. Every time you find yourself writing the same lines twice, that's a signal to extract a function.

1. Defining and Calling a Function

def greet():
    print('Hello!')
    print('Nice to meet you.')

# Call the function — runs everything inside
greet()   # Hello! / Nice to meet you.
greet()   # called again — same output, no copy-paste!
💡 Tip: The def line only defines the function — it doesn't run it yet. The code inside only runs when you call the function by writing its name followed by ().

2. Parameters and Arguments

def greet(name):          # 'name' is a parameter
    print(f'Hello, {name}!')

greet('Alice')            # 'Alice' is the argument
greet('Bob')
# Hello, Alice!
# Hello, Bob!

def add(a, b):             # two parameters
    print(a + b)

add(3, 5)                 # 8
add(10, 20)               # 30

3. Return Values

def add(a, b):
    return a + b       # sends the result back to the caller

result = add(3, 5)    # store the returned value
print(result)         # 8
print(add(10, 20))    # 30 — use directly in print

# Without return, a function gives back None
def say_hi():
    print('Hi!')

value = say_hi()       # Hi! is printed
print(value)           # None
💡 Tip: A function with return gives back a value you can store or use in calculations. A function without return gives back None — useful for functions that just print or do something, not compute something.

4. Default Parameter Values

def greet(name='friend'):   # 'friend' is the default
    print(f'Hello, {name}!')

greet()            # Hello, friend!  (default used)
greet('Alice')     # Hello, Alice!   (default overridden)

def power(base, exp=2):     # exp defaults to 2
    return base ** exp

print(power(3))     # 9  — 3 squared
print(power(3, 3))  # 27 — 3 cubed

5. Variable Scope: Local vs Global

def calculate():
    result = 42        # local variable — only exists inside
    print(result)      # 42 — works fine here

calculate()
# print(result)     # ERROR! result is not defined here

# Global variables (defined outside) are readable inside
pi = 3.14

def circle_area(r):
    return pi * r * r  # pi is global, we can read it

print(circle_area(5))  # 78.5
💡 Tip: Variables created inside a function only exist there — you cannot use them outside. This is called local scope. It's a feature, not a bug: functions are self-contained so they don't accidentally mess with the rest of your program.

💡 Hint
Use def say_hello(): then indent the body with 4 spaces. Call it with say_hello().

Task 2 — Greeting with a Name

Write a function greet(name) that prints "Hello, [name]! Nice to meet you.". Call it with three different names.

💡 Hint 1
Define the function with one parameter: def greet(name):.
💡 Hint 2
Inside the function use an f-string: print(f"Hello, {name}! Nice to meet you.").

Task 3 — Calculator Function

Write a function multiply(a, b) that returns the product of a and b. Print the result of calling it with several different pairs of numbers.

💡 Hint
Use return a * b inside the function, then print(multiply(3, 4)) outside.

Task 4 — Default Greeting

💡 Hint 1
Set the default in the definition: def greet(name="friend"):.
💡 Hint 2
Call it both ways: greet() and greet("Alice") to see the difference.
