✏️ Homework 3: Cycles (Loops)

Complete all tasks below. Write your code in a file called homework_3.py and run it to check your answers.

💡 Reminder: Use for i in range(...) for a fixed number of repetitions, for item in list to go through a list, and while condition when you don't know how many times in advance. Use break to stop early and continue to skip an iteration.

🎯 Tasks

Task 1 — Multiplication Table

Ask the user to enter a number. Print the full multiplication table for that number (from 1 to 10).

Example output for 7:

7 x 1 = 7
7 x 2 = 14
...
7 x 10 = 70

Task 2 — Find the Biggest

Ask the user to enter 5 numbers one by one (using a loop).
After all 5 are entered, print the largest number.
Do not use the built-in max() function — find it yourself!

Task 3 — FizzBuzz Extended

Print all numbers from 1 to 50. But replace:

  • multiples of 3 with "Fizz"
  • multiples of 5 with "Buzz"
  • multiples of both 3 and 5 with "FizzBuzz"

Count how many "Fizz", "Buzz", and "FizzBuzz" appear, and print the counts at the end.

Task 4 — Shopping List

Use a while loop to let the user add items to a shopping list one by one.
Stop when the user types "done".
After they are done, print the full list with numbered items and the total count.

Task 5 — Diamond Pattern ⭐

Use nested loops to print a diamond of stars. For size 3 the output should be:

  *
 ***
*****
 ***
  *

Ask the user for the size (the middle row will have 2 * size - 1 stars).
Hint: split it into two parts — the top half (including the middle) and the bottom half.
