📖 Lecture 3: Cycles (Loops)

A loop lets you run the same block of code many times without repeating yourself. Python has two types of loops: for and while.

1. The for Loop

Use a for loop to repeat something a fixed number of times or to go through a list.

# Counting from 0 to 4
for i in range(5):
    print(i)
# Prints: 0  1  2  3  4

# range(start, stop) — from start up to (but not including) stop
for i in range(1, 6):
    print(i)
# Prints: 1  2  3  4  5

# range(start, stop, step) — every 2nd number
for i in range(0, 10, 2):
    print(i)
# Prints: 0  2  4  6  8

2. Looping Over a List

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
# apple
# banana
# cherry

# Useful: loop with both index and value using enumerate()
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")
# 0: apple
# 1: banana
# 2: cherry

3. The while Loop

A while loop keeps running as long as a condition is True.

count = 1
while count <= 5:
    print(count)
    count += 1   # count = count + 1
# Prints: 1  2  3  4  5

# Asking until the user enters the right answer
secret = "python"
guess = ""
while guess != secret:
    guess = input("Guess the secret word: ")
print("Correct! 🎉")
⚠️ Infinite loop! If the condition in a while loop never becomes False, the program will run forever. Always make sure something changes inside the loop!

4. break and continue

# break — stop the loop immediately
for i in range(10):
    if i == 5:
        break
    print(i)
# Prints: 0  1  2  3  4

# continue — skip the rest of this iteration, go to the next one
for i in range(10):
    if i % 2 == 0:
        continue   # skip even numbers
    print(i)
# Prints: 1  3  5  7  9

5. Nested Loops

A loop inside another loop — useful for working with tables or patterns.

# Multiplication table (3x3)
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i*j}")
    print()   # blank line after each row

# Drawing a triangle with stars
for i in range(1, 6):
    print("*" * i)
# *
# **
# ***
# ****
# *****

🎯 Tasks

Task 1 — Count Down

Use a for loop to print numbers from 10 down to 1, then print "Blast off! 🚀".
Hint: range(10, 0, -1) counts backwards.

Task 2 — Sum of Numbers

Ask the user to enter a number n. Use a loop to calculate the sum of all numbers from 1 to n and print the result.
Example: if n = 5, print "Sum = 15" (1+2+3+4+5).

Task 3 — Favourite List

Create a list of 5 of your favourite things (movies, foods, games — your choice!).
Use a for loop to print each one with its position number.
Example: "1. Pizza", "2. Minecraft", …

Task 4 — Guess the Number

Pick a secret number (hardcode it in your script). Use a while loop to keep asking the user to guess. After each guess, print "Too high!", "Too low!", or "Correct! 🎉". Stop when they guess right.

Task 5 — Star Triangle

Use a nested loop to print a 5-row right-aligned triangle of stars:

    *

*****

Hint: each row needs spaces before the stars. The number of spaces decreases as the row grows.
